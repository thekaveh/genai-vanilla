# Rebrand Plan: GenAI Vanilla Stack → Atlas Stack (ATLAS)

## Executive Summary

This project originally launched as **GenAI Vanilla Stack** — a modular AI development toolkit. Today it has evolved into a multi-track infrastructure platform supporting:

- AI Engineering (original)
- Data Engineering
- ML Engineering
- MLOps/GitOps
- Observability pipelines
- And more tracks to come

The old name no longer reflects the project's scope. **Atlas** — the Titan who bore the heavens — is the perfect metaphor: this stack bears the weight of an entire AI/ML/ data engineering foundation, holding up the globe for everything built on top of it.

## Phase 3: Recommended Logo Choice

### Decision Matrix

| Sketch | Pros | Cons |
|--------|------|------|
| **A1 (doom font)** | Bold, authoritative, clean, 32 chars wide | Pure text, no visual symbol |
| **A3 (3-d font)** | Dramatic depth, visually striking | 52 chars wide — needs wider terminal or line wrapping |
| **A5 (gothic)** | Mythical feel, ornate | 48 chars, may look "too dark" in a blue palette |
| **B1 (JP2A 60×30)** | Your Atlas image preserved, literal Titan metaphor | 60 chars — fits but leaves margin; gradient rendering needs care |
| **B2 (JP2A 80×36)** | Most detail, globe visible | 80 chars — requires full-width terminal |
| **D1 (Atlas figure + ATLAS text)** | Best of both: literal symbol + clear project name | Tallest at ~37 rows; needs layout planning in wizard |

### Recommendation

**Primary logo: D1 (Atlas figure + ATLAS text stacked)** — it's the most thematically complete. The image preserves your Atlas artwork, and the ATLAS text (doom font) gives a clean secondary lockup.

**Fallback for narrow screens: A1 (doom font only)** — if the terminal is too narrow for the full art, fall back to pure ATLAS in doom font.

## Phase 4: Color Palette Considerations

Current `LOGO_GRADIENT` in `palette.py`:
```python
LOGO_GRADIENT = ["#4a9eff", "#7dcfff", "#a8d4e6", "#7dcfff"]  # 4-stop
```

This is a blue gradient (deep navy → bright cyan). For Atlas, I suggest a **sky-to-night** gradient that preserves the blue family but adds a sense of depth:

| Stop | Current | Suggested | Rationale |
|------|---------|-----------|-----------|
| 1 (top) | `#4a9eff` | `#5ba3ff` | Keep it — bright sky blue |
| 2 | `#7dcfff` | `#4a90e0` | Slightly deeper blue |
| 3 | `#a8d4e6` | `#2d6cb6` | Deeper — ocean blue |
| 4 (bottom) | `#7dcfff` | `#0e3d6a` | Deepest — twilight navy |

**Why**: Atlas bears the *heavens* — the gradient should feel like it's rising from the depths of the ocean up to open sky. The current gradient ends bright at the bottom; reversing it so it deepens toward the bottom feels more grounded, like Atlas himself.

The `ACCENT_BAR_STOPS` vertical gradient (`[#4a9eff, #7dcfff, #98c379, #7dcfff, #565f89]`) can stay as-is since it's a general-purpose accent bar, not the logo.
